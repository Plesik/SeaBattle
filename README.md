# SeaBattle

SeaBattle — это локальная игра «Морской бой» для двух игроков на одном компьютере, написанная на C++ с использованием SFML.

## О проекте

Проект представляет собой настольную версию классического «Морского боя» с графическим интерфейсом.  
Игроки по очереди:

- расставляют свои корабли на поле 10×10;
- передают ход друг другу;
- делают выстрелы по полю противника;
- пытаются первыми уничтожить весь флот соперника.

Игра запускается в одном окне и предназначена для двух человек за одним компьютером.

## Возможности

- главное меню;
- экран с правилами;
- локальная игра для двух игроков;
- поэтапная расстановка кораблей;
- пошаговый режим боя;
- графический интерфейс на SFML;
- кастомные кнопки интерфейса.

## Технологии

- C++
- SFML
- CMake

## Сборка и запуск

Проект собирается на Linux (Ubuntu) через CMake.

### Установка зависимостей

```bash
sudo apt update
sudo apt install -y \
    build-essential cmake ninja-build git \
    libxrandr-dev libxcursor-dev libxi-dev libudev-dev \
    libfreetype-dev libflac-dev libvorbis-dev \
    libgl1-mesa-dev libegl1-mesa-dev
```

### Сборка

```bash
cmake -B build -G Ninja
cmake --build build
```

### Запуск

```bash
./build/SeaBattle
```

## Управление

### В меню
- **Play** — начать игру
- **Rules** — открыть экран с правилами
- **Exit** — выйти из приложения

### Во время игры
- управление осуществляется мышью;
- игроки по очереди расставляют корабли;
- затем по очереди выбирают клетки для выстрелов.

> При необходимости этот раздел можно расширить после финальной доработки логики расстановки и стрельбы.

## Правила игры

В проекте реализована классическая схема «Морского боя»:

- 2 игрока;
- поле 10×10;
- у каждого игрока:
  - 1 линкор на 4 клетки;
  - 2 крейсера на 3 клетки;
  - 3 эсминца на 2 клетки;
  - 4 подлодки на 1 клетку.

Цель игры — первым уничтожить все корабли противника.
